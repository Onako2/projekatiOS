# projekatiOS

projekatiOS is just a small website for random stuff I'll do in the future. Currently I have no idea what apps I could add :(  
In case you have any idea, you can PR it :)  

Current apps:

* Scratch (embedder, powered by Turbowarp)
* [MegaSkins](https://github.com/Onako2/MegaSkins/) (website that associates Minecraft skins with a machine readable description)
* Blåhaj Music, listen to beautyful music by Blåhaj sharks
* Front Page, the simple nuc.de.majic.rs page
* Custom, embed your own web applications into projekatiOS (btw. you can also embed projekatiOS, is it called projekatiOS²?)

## Special features

* Minimize
* Reload
* Focus managment (click the window's head bar)
* A nice panorama from the city of Bamberg

## Deployment

This project is deployed on <https://nuc.de.majic.rs/projekatios/>

## Development

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. And tada!
